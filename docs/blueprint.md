# **App Name**: Bazar Moçambique AI

## Core Features:

- Recomendação de Produtos por IA: Ferramenta de IA para recomendar produtos com base nas preferências do usuário e no histórico de navegação, aprimorando a experiência de compra. Esta ferramenta irá usar raciocínio para decidir quais informações usar.
- Listagem e Exibição de Produtos: Exiba produtos de 'Quadros Artísticos' e 'Smart Tag - Rastreador', com informações detalhadas, como preço, descrição e estoque disponível.
- Realização de Pedidos: Modal de checkout para obter detalhes do usuário.
- Confirmação de Pedido: Página de confirmação após a realização bem-sucedida do pedido.

## Style Guidelines:

- Cor primária: Azul profundo (#004494), inspirado no logotipo da marca, sugerindo confiança e confiabilidade.
- Cor de fundo: Azul muito claro (#F0F8FF), um tom dessaturado da cor primária que confere uma sensação limpa e arejada à interface.
- Cor de destaque: Laranja vibrante (#FF9933), para destacar elementos interativos e botões de chamada para ação, garantindo que se destaquem.
- Fonte do corpo e título: 'Inter', sans-serif, uma fonte moderna e neutra para clareza de leitura e uma aparência contemporânea.
- Ícones do Font Awesome: Sacola de compras, estrela, círculo de verificação, seta para a esquerda, etc. Fornecem um guia visual para as funções do aplicativo
- Layout baseado em grade para listagens de produtos para garantir uma apresentação limpa e organizada.
- Animações de carregamento com um spinner sutil para indicar processos em andamento.
